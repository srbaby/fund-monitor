# 归档层（考古用，不参与日常阅读）

某节规范被整体取代时，**把旧版移到这里，不要直接删**。命名：`<原文件名>-<YYYY-MM-DD>.md`。

**为什么留着**：被删掉的规范里常常藏着"当初为什么否决另一种做法"。
那部分理由一旦随文件消失，后来者会重新提出同一个被否决过的方案，并且理直气壮。

**给 AI**：日常任务不需要读这里。只有在以下情况才翻：
- 想推翻一条现有设计，需要先确认它当初是怎么来的；
- 发现代码里有文档解释不了的东西，怀疑是历史遗留；
- 用户说"以前不是这样的"。

配套的决策理由在 [../DECISIONS.md](../DECISIONS.md)，那份是主线，这里是原始文本备份。
